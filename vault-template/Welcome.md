# Welcome to your Second Brain

This vault was generated by **Argus** (`argus init`). It follows a PARA-inspired
layout tuned so both you *and* your assistant can read it.

## Map

| Folder | Purpose |
| --- | --- |
| `00-Inbox/` | Quick capture — Argus triages this for you |
| `10-Daily/` | Daily notes: agenda, log, reflections, tasks |
| `15-Courses/` | One folder per course — notes, uploaded materials, Argus study output |
| `20-Projects/` | One note per active project (goal, next actions) |
| `30-Areas/` | Ongoing responsibilities: health, finances, learning |
| `40-People/` | One note per person; Argus links mentions |
| `50-Reference/` | Evergreen knowledge notes |
| `99-Private/` | **Never** indexed and never sent to any AI |

## Conventions

- Tasks use the Obsidian Tasks syntax: `- [ ] Renew passport 📅 2026-07-20 ⏫ #areas/admin`
- Every note carries `type`, `created`, `tags` frontmatter.
- Tag a note `#no-ai` to keep it out of Argus's index entirely.
